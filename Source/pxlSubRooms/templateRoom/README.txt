Sub-Rooms are similar to Rooms,
However their info, assets, scripts, etc. are loaded into an existing room.

Currently, sub-rooms are expected to load into the main environment within pxlNav
The Time Sqaure Canyon.

 -- -- -- -- --

Info for creating Sub-Rooms,
Have the same prep, notations, and function steps, as normal Room dev.

Please see ./src/js/pxlRooms/README.txt for more info on room creation.
